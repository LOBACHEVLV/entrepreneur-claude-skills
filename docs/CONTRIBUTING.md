# Contributing

Thanks for your interest in contributing to Entrepreneur Claude Skills.

## Submitting a New Skill

1. **Fork** this repository
2. **Create a skill folder** under the appropriate category in `skills/`
3. **Follow the skill format** documented in [SKILL_FORMAT.md](SKILL_FORMAT.md)
4. **Include at minimum:**
   - `SKILL.md` with YAML frontmatter and complete workflow
   - `README.md` with description, usage examples, and expected output
5. **Open a pull request** with a clear description of what the skill does and why it belongs here

## Skill Quality Standards

This is a curated collection. Skills must meet these criteria:

- **Solves a real problem** — not a toy prompt. Could a founder use this today?
- **Structured workflow** — clear steps, not just "write me X." The skill should guide the agent through a framework.
- **Tested output** — include at least one example showing real input and expected output
- **Specific domain** — one skill, one job. Don't combine unrelated workflows.
- **Production-ready** — works reliably, handles edge cases, produces useful output on first run

## Improving Existing Skills

Found a skill that could be better? PRs welcome for:

- Better frameworks or references
- Additional examples
- Clearer activation triggers
- Bug fixes in workflow steps

## Reporting Issues

Open an issue if a skill:
- Produces poor output for a common use case
- Has unclear instructions
- Is missing from a category where it clearly belongs

## Code of Conduct

Be helpful, be specific, be respectful. We're all building here.
